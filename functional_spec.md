# 기능 명세서 (Functional Specification) - 캠멍 (CAM-MUNG)

본 문서는 **캠멍 (CAM-MUNG)** 애플리케이션의 핵심 기능과 브랜드 철학을 정의합니다. 모든 코드 수정 및 리팩토링은 본 명세서를 준수해야 하며, 명기된 브랜드 아이덴티티가 훼손되어서는 안 됩니다.

## 🧘 브랜드 철학 (Brand Philosophy)
1. **캠핑 + 멍 (불멍/물멍)**: 목적 없이 정보를 훑어보는 행위 자체를 휴식으로 정의. 캠핑의 꽃인 '멍 때리기'를 앱 사용 경험으로 가져왔습니다.
2. **데이터 안심 (No Data Stress)**: 실시간 검색으로 데이터 소모를 걱정할 필요 없이, 내 폰에 저장된 로컬 DB로 '안심하고 멍하니' 즐길 수 있다는 신뢰감을 줍니다.
3. **직관적인 위치 기반**: 내 주변(집, 회사)의 거리와 비용을 즉각 보여주어, 복잡한 계산 없이 멍하니 보고만 있어도 갈 만한 곳이 추려진다는 편리함을 담았습니다.

---

## 🏗️ 시스템 아키텍처 (Architecture)
- **Frontend**: React (Vite) + Tailwind CSS 기반의 모던 웹 앱.
- **Data Manager**: `useCampData.ts` 훅을 통한 로컬 JSON DB 핸들링.
- **Recovery**: Git 기반의 소스 자동 복구 시스템 (`recover_app.bat`).

---

## 📋 핵심 기능 명세 (Feature Spec)

### 1. 검색 및 필터링 (Search & Filter)
- **통합 검색**: 캠핑장 이름, 주소, 지역명 실시간 검색.
- **2단계 지역 필터**: 
    - [도] 선택 -> [시군구] 칩 목록 동적 생성 및 필터링.
- **시설 및 테마 태그**: 
    - `#`으로 시작하는 태그 검색 시 `fac`(시설), `env`(환경), `type`(유형) 필드 전역 교차 검색.
- **거리 슬라이더 (Distance Slider)**: 
    - 주소 분석 완료 전까지는 비활성화(Lock) 및 안내 문구 노출.
    - 분석 완료 후 실시간 소요 시간/거리 기반 필터링.
- **가격 슬라이더**: 
    - 최대치(20만원) 설정 시 '제한 없음'으로 처리하여 모든 가격대 허용.

### 2. 검색 결과 뷰 (Results View)
- **실시간 요약**: 상단에 `총 000개 중 000개 발견` 문구 상시 노출.
- **정보 카드 (CampCard)**:
    - 가격 표시: `Price From` 라벨 병기 및 이탤릭체 강조.
    - 소요 시간: 홈(HOME) 및 회사(WORK) 기준 배지 분리 표시.
    - 예약 링크: 플랫폼별(캠핏, 땡큐, 네이버 등) 전용 버튼 생성.

### 3. 데이터 및 버전 관리 (Version Control)
- **로컬 내보내기/불러오기**: 캠핑 데이터(JSON)를 파일로 저장 및 복구.
- **GitHub 롤백**: 클라우드 저장소의 특정 SHA 시점으로 DB 복구 기능.
