# 기능 명세서 (Functional Specification) - Camping-Sync

본 문서는 Camping-Sync 애플리케이션의 핵심 기능과 UI 요소를 정의합니다. 모든 코드 수정 및 리팩토링은 본 명세서를 준수해야 하며, 명기된 기능이 누락되거나 변형되어서는 안 됩니다.

## 1. 전역 내비게이션 (Tabs)
- [ ] **Search (검색 설정)**: 지역, 거리, 가격, 시설 등의 필터를 설정하는 메인 화면.
- [ ] **Results (검색 결과)**: 설정된 필터에 따라 필터링된 캠핑장 리스트를 보여주는 화면. (실시간 개수 표시 포함)
- [ ] **Favorites (찜 목록)**: 사용자가 '찜'한 캠핑장만 모아서 보여주는 화면.

## 2. 검색 및 필터링 (Search & Filters)
- [ ] **통합 검색창 (Search Bar)**: 
    - 상단에 고정된 눈에 띄는 디자인.
    - 캠핑장 이름, 주소, 지역명(도/시군구)을 실시간으로 검색.
- [ ] **지역 필터 (Do Filter)**: '전체'부터 각 도별(경기, 강원 등) 칩 형태의 선택 UI.
- [ ] **집에서의 거리/시간 (Distance Slider)**:
    - OSRM 엔진을 통한 '분' 단위 소요 시간 기준 필터링.
    - **분석 전**: 슬라이더 조작 가능하되, 결과 반영은 분석 완료 후에만 작동 (또는 잠금 UI 유지).
    - **분석 후**: 설정한 시간(분) 이내의 캠핑장만 결과에 표시.
- [ ] **1박 가격 (Price Slider)**:
    - 0원에서 200,000원+ 까지 조작 가능한 슬라이더.
    - 20만 원 이상은 '제한 없음'으로 처리하여 모든 가격대 포함.
- [ ] **시설 및 테마 (Facilities & Tags)**:
    - 전기, 와이파이, 온수 등 필수 시설 다중 선택.
    - #차박, #반려동물 등 감성 태그 필터링.

## 3. 캠핑장 카드 UI (Camp Card)
- [ ] **이미지**: 캠핑장 사진 표시 ( 없을 시 대체 이미지).
- [ ] **찜하기 버튼**: 카드 우측 상단에 위치, 클릭 시 즉시 Favorites 탭에 반영.
- [ ] **소요 시간 배지 (Time Badges)**:
    - 분석 완료 시 'HOME 00분', 'WORK 00분' 형태의 배지 표시.
- [ ] **기본 정보**: 캠핑장 이름, 상세 주소, 업종(유형) 표시.
- [ ] **가격 정보**: 좌측 하단에 'Price From'과 함께 금액 표시.
- [ ] **액션 버튼**:
    - **네이버 지도**: 클릭 시 네이버 지도 검색 결과로 이동.
    - **예약하기**: 캠핏, 땡큐캠핑, 네이버예약 등 플랫폼별 라벨링 및 이동.

## 4. 데이터 및 분석 (Data & Analysis)
- [ ] **좌표 설정**: 설정 모달에서 집/회사 주소를 입력하고 좌표를 추출.
- [ ] **OSRM 분석**: 전체 캠핑장 데이터에 대해 실시간 경로 분석 수행.
- [ ] **자동 업데이트**: 앱 실행 시 GitHub의 최신 DB 버전 체크 및 자동 다운로드.
