# 마스터 블루프린트 (Master Blueprint) - 캠멍 (CAM-MUNG) 🧘

> 본 문서는 **캠멍 (CAM-MUNG)** 프로젝트의 가장 상위 기획서이며, 모든 개발과 철학의 기준이 됩니다.

## 1. 프로젝트 비전 (Vision)
"캠핑의 꽃인 '멍 때리기'를 디지털 경험으로 승화시킨다."
단순한 정보 제공을 넘어, 사용자가 아무런 스트레스 없이 멍하니 최고의 캠핑장을 발견하고 안심하며 즐길 수 있는 환경을 구축하는 것이 목표입니다.

## 2. 핵심 브랜드 철학 (The 3 Mung)
1. **캠핑 + 멍 (Relax)**: 
    - 목적 없는 정보 탐색 자체를 휴식으로 정의. 
    - UI는 최소한의 조작으로 최대한의 정보를 '멍하니' 볼 수 있도록 설계.
2. **데이터 안심 (No Data Stress)**: 
    - 실시간 검색으로 인한 데이터 소모 걱정 NO! 
    - 내 폰에 저장된 로컬 DB로 '안심하고 멍하니' 즐길 수 있는 신뢰 플랫폼.
3. **직관적인 위치 기반 (Convenience)**: 
    - 집/회사 기준의 거리와 비용을 즉각 노출. 
    - 복잡한 필터 조작 없이도 갈 만한 곳이 자동으로 조준되는 편리함.

## 3. 마스터의 엄격한 선별 기준
- **필수 조건**: 50,000원 이하 (가성비 최우선)
- **우선 순위**: 파쇄석 사이트, 사이트 옆 주차, 조용한 환경
- **실시간성**: 원본 71KB 로직을 통한 빈자리 상태(예약 가능 여부) 감별

## 4. 기술적 요구 사항 (Technical Stack)
- **Frontend**: React (Vite) / Tailwind CSS / Material Design 3
- **Data Engine**: Local JSON DB + Git Baseline SVN 보호 체계
- **Automation**: GitHub Actions를 통한 10분 주기 스크래핑/감시 연동 (Python/Scraper)
- **Alert**: 텔레그램을 통한 "마스터, 자리가 났습니다!" 즉시 통보 시스템
